AbiturientPro – Приложение для автоматизации приёмной комиссии колледжа
--- О приложении ---

Мобильное приложение для автоматизации работы приёмной комиссии ГБПОУ «Тверской промышленно-экономический колледж». Приложение предназначено для абитуриентов, их родителей и сотрудников приёмной комиссии. Оно позволяет подавать документы, отслеживать статус заявлений, подтверждать их (родителями для несовершеннолетних и администратором) и формировать списки поступающих с фильтрацией по специальностям.

--- Функциональность ---

Для всех пользователей:

Вход в приложение по логину и паролю.

Настройка темы (светлая/тёмная/системная) и языка (русский/английский).

Боковое навигационное меню с пунктами, доступными согласно роли.

Для абитуриента:

Просмотр приветственного экрана с персональным обращением.

Подача заявления на выбранную специальность (список специальностей взят с официального сайта колледжа).

Просмотр списка своих заявлений с отображением текущего статуса (ожидает подтверждения родителя / ожидает подтверждения администратора / зачислен).

Для родителя:

Просмотр списка детей, привязанных к аккаунту.

Просмотр заявлений каждого ребёнка.

Подтверждение заявлений несовершеннолетних абитуриентов (обязательное условие для дальнейшего рассмотрения администратором).

Для администратора:

Просмотр всех поданных заявлений с возможностью фильтрации по специальностям.

Подтверждение зачислений после родительского согласия (если абитуриент младше 18 лет).

Управление статусами заявлений.

--- Технические особенности ---

Язык программирования: Java

Минимальная версия Android: API 21 (Android 5.0)

Используемые библиотеки:

AndroidX AppCompat

Material Components

ConstraintLayout

Navigation (для фрагментов)

Preference (для настроек)

Архитектура: хранение данных в памяти (MockRepository) для демонстрации; при необходимости легко заменить на реальную базу данных.

--- Установка и запуск ---

Клонируйте репозиторий:
  git clone https://github.com/Zuzzy62/AbiturientPro.git

Откройте проект в Android Studio.

Дождитесь синхронизации Gradle (при необходимости установите требуемые версии плагинов).

Соберите проект: Build → Make Project.

Запустите приложение на эмуляторе или реальном устройстве (подключите устройство с включённой отладкой по USB).
